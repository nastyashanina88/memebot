# Meme Bot

Бот парсит Telegram-каналы, присылает найденные мемы тебе в личку на одобрение и публикует одобренные в твой канал по расписанию.

---

## Что умеет

- Парсит **28 источников** параллельно каждый час
- Фильтрует рекламу, ссылки, длинные подписи автоматически
- Присылает мемы в личку с кнопками — одобрить, пропустить, добавить подпись или запостить сразу
- Публикует по расписанию **7–10 раз в день** с 9:00 до 22:00 МСК
- Уведомляет после каждой публикации с остатком очереди
- Сохраняет картинки в базе — мем опубликуется даже если CDN-ссылка протухла

## Команды

| Команда | Что делает |
|---------|------------|
| `/fetch` | Проверить каналы прямо сейчас и получить новые мемы |
| `/post` | Опубликовать следующий мем из очереди вручную |
| `/schedule` | Расписание публикаций на сегодня с обратным отсчётом |
| `/queue` | Сколько мемов ждут публикации |
| `/status` | Статистика базы (новые, одобренные, пропущенные, опубликованные) |
| `/skip` | Одобрить мем без подписи (когда бот ждёт текст) |
| `/clearsent` | Сбросить все непросмотренные мемы — чистый лист перед `/fetch` |
| `/clearqueue` | Удалить из очереди битые посты без картинки |
| `/help` | Показать все команды прямо в боте |

## Кнопки при одобрении

| Кнопка | Действие |
|--------|---------|
| ✅ | Добавить в очередь без подписи |
| 🚀 | Опубликовать прямо сейчас |
| ✍️ | Добавить в очередь с подписью |
| ❌ | Пропустить |

## Установка

### 1. Зависимости

```bash
pip install -r requirements.txt
```

### 2. Создай бота

1. Напиши [@BotFather](https://t.me/BotFather) → `/newbot`
2. Получи `BOT_TOKEN`
3. Добавь бота в свой канал как **администратора** с правом публикации

### 3. Заполни .env

```
BOT_TOKEN=ваш_токен
MY_CHANNEL=@ваш_канал
ADMIN_CHAT_ID=ваш_telegram_id  # необязательно, можно вместо этого написать /start боту
```

### 4. Запуск

```bash
python bot.py
```

Напиши `/start` боту в личку — он зарегистрирует твой чат и начнёт присылать мемы.

## Настройки (bot.py)

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| `SOURCE_CHANNELS` | 28 каналов | Откуда брать мемы |
| `POSTS_PER_DAY_MIN/MAX` | 7–10 | Сколько постов в день |
| `POST_START_HOUR` | 9 | С какого часа (МСК) |
| `POST_END_HOUR` | 22 | До какого часа (МСК) |
| `MAX_CAPTION_LEN` | 150 | Максимальная длина подписи |
| `FETCH_INTERVAL` | 3600 | Как часто проверять каналы (сек) |
| `FETCH_HOURS_BACK` | 72 | Брать посты за последние N часов |
| `MAX_SEND_PER_FETCH` | 25 | Максимум мемов на одобрение за один /fetch |

## Деплой на Railway

1. Создай сервис из GitHub-репозитория
2. Добавь переменные окружения: `BOT_TOKEN`, `MY_CHANNEL`, `ADMIN_CHAT_ID`
3. Добавь **Volume** с Mount Path `/data` и переменную `DATA_DIR=/data` — база не будет теряться при редеплоях
